# Docker Backend

## Clone

```
git clone --recurse-submodules --remote-submodules git@github.com:Ujian-Online/docker-backend.git
```

## Start Docker

```
docker-compose up -d
```

## Stop Docker

```
docker-compose down
```

## MySQL Database

- Agar file Database tidak hilang, docker akan membuat folder data, sehingga ketika di stop, data tidak hilang
